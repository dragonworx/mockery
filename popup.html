Setup Guide
  1. Start the server: npm start or node mock-server.js
  2. Manage mocks via the file system:
    • Add mock files to the mocks/ folder
    • Create folders to organize your mocks
    • Edit .mocks.json to add URL patterns and file paths
    • Example: {"pattern": "api/users", "file": "mocks/users.json"}
  3. Toggle the extension on — matched requests will serve your files
  4. Files in the mocks folder can be edited directly and changes are live!

Setup Guide
  1. Start the server: npm start or node mock-server.js
  2. Add rules via this UI:
    • Enter a URL pattern to match
    • Option A: Click "Choose file..." to select a file from your computer. The extension will automatically read and save it to the mocks folder.
    • Option B: Enter a file path (relative paths like mocks/data.json or absolute paths work)
    • New: No need for absolute paths anymore! Selected files are read directly and saved to mocks/ with unique names
  3. Toggle the extension on — matched requests will serve your files
  4. Files in the mocks folder can be edited directly and changes are live!

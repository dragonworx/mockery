Setup Guide
  1. Create a .mocks.json file in your project directory
  2. Add rules mapping URL patterns to local files: 
    [
  {
    "pattern": "address-book\\.json",
    "file": "./stubs/address-book.json",
    "isRegex": true
  }
]
  3. Start the server: node mock-server.js
  4. Toggle the extension on — matched requests will serve your local files
  5. Edit stub files in place — changes are picked up on next request
